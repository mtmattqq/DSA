```pasudo
Algorithm FindDuplicate(int array A[], int n)
    int array B[](n, 0)
    for i <- 1 to n do
        B[A[i]] += 1
        if B[A[i]] > 1 then 
            return A[i]
    return NOT_FOUND
```

This algorithm has time complexity $O(n)$, which is the best for this problem. We use an addtional array B, where 

$$
    B[i] := \text{the number of apperence of i.}
$$
